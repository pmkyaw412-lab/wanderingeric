Every thesis has a graveyard of paper notebooks with half-legible run parameters. weld-log is my way out: one command records a run (current, voltage, travel speed, coupon ID), and one command plots any parameter against any other.

Nothing clever — just the discipline of structured data applied to lab life. The repo link will appear here once I've cleaned it up enough to not be embarrassing.
